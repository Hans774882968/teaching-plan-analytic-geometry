椭圆的定义与基本性质

📏 椭圆的定义

椭圆是平面上到两个定点（焦点）的距离之和等于常数的点的集合。这两个定点称为焦点，常数必须大于两焦点之间的距离。

数学表达式：对于任意点P在椭圆上，有 |PF₁| + |PF₂| = 2a，其中F₁和F₂是焦点，2a是常数。

📐 椭圆的标准方程

当椭圆中心在原点，焦点在x轴上时，标准方程为：

\[\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\]

其中：

  • a - 半长轴长度 (a > b)
  • b - 半短轴长度
  • c - 焦距的一半，满足 \(c = \sqrt{a^2 - b^2}\)

🔍 椭圆的重要性质

1. 离心率：e = \frac{c}{a} (0 < e < 1)，表示椭圆的扁平程度

2. 焦点性质：从椭圆一个焦点发出的光线，经椭圆反射后会经过另一个焦点

3. 对称性：椭圆关于长轴、短轴和中心对称

4. 面积公式： S = \pi \times a \times b

💡 椭圆二级结论

焦点三角形

椭圆上一点P与两个焦点F₁、F₂构成的三角形称为焦点三角形：

  • 周长：2a + 2c
  • 当P位于短轴端点时，∠F₁PF₂最大
  • 面积：\(S = b^{2} \times \tan\left(\frac{\theta}{2}\right)\) ，其中θ为∠F₁PF₂

直径性质

平行弦的中点轨迹：椭圆的一组平行弦中点的轨迹是过椭圆中心的线段（直径）

光学性质

从椭圆一个焦点发出的光线，经椭圆反射后必经过另一个焦点。这一性质在卫星天线和声学设计中应用广泛。
